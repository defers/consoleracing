Напишите консольное приложение на языке Java, которое симулирует гонки.
В гонках участвуют разные типы транспортных средств: грузовики, легковушки.
Все транспортные средства стартуют одновременно.
После завершения заезда, приложение выводит таблицу первенства.
У каждого из транспортных средств есть своя скорость и вероятность прокола колеса. Когда колесо прокалывается,
транспортное средство останавливается и сообщает о проколе в консоль.
Перед началом движения каждое транспортное средство пишет в консоль свои параметры.
